Jupyter 环境配置

安装 Python 与 Jupyter
Mac 下用 brew 安装即可

brew install python jupyter

需要安装的 VSC 插件

  • Python
  • VS Code Jupyter Notebook Previewer

Notebook 创建与运行

打开一个空文件夹，Cmd+Shift+P 呼出命令面板
输入 python create 进行搜索，选择 Python: Create New Blank Jupyter Notebook
