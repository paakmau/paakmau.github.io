## Python 环境配置

xjb 装就行了<br>
`macOS` Catalina 自带的 Python3 不建议用，`brew install python3`

## 需要安装的 VSC 插件

- Python

## 工作区配置

打开一个空文件夹，在其中新建并编辑一个文件 `hello.py`，如下

```py
print("Hello World")
```

按 `F5` 运行，配置选 `Python`

另外，可以根据提示装一下语法提示器
